Piltide hindamine





Eesmärk: Veebilehe eesmärk on piltide hindamine. Kasutaja saab lehel üles laadida pildi. Kõik pildid kuvatakse seejärel ühel lehel. Kõike pilte saab hinnata.

Sihtrühm: Piltide hindamise konkursid,fotograafid jpm.








Andmebaasi tabeli loomise lause:

CREATE TABLE piltide_hindamine(
Id int(20) PRIMARY KEY AUTO_INCREMENT,
Reiting float,
Photoid varchar(200)
);

Kokkuvõte:

Saime kõvasti kogemust juurde, kuidas PHP-koodi kirjutada. Pildi üleslaadimisel pilt ei kuva avalehel. Keeruline oli reitingute andmebaasi sisestamine.
